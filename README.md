## How to Edit This Code

### Using Your IDE Locally

1. Clone the repository:

```bash
git clone git@github.com:Maheesh09/SE-Project-Demo-Frontend.git

Navigate to the project folder:

cd SE-Project-Demo-Frontend

Install dependencies:

npm install

Start the development server:

npm run dev
Edit Files Directly on GitHub

Navigate to the file you want to edit.

Click the "Edit" button (pencil icon) in GitHub.

Make your changes and commit.

Use GitHub Codespaces

Open the repository main page.

Click the green "Code" button → Codespaces → New codespace.

Edit files directly and commit.

Project Features

Student registration & login (email + Google OAuth)

Profile management & mandatory profile completion

Dashboard: XP, streaks, daily goals, weekly activity

Courses: lessons, quizzes, textbooks, past papers

Quizzes: topic-wise & term-wise

Analytics: skill radar, score trends, daily study time

Leaderboard: district & national ranking

AI Tutor: interactive help for subjects
